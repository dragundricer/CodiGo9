Se desea crear una pagina web de emision de notas de pedido de un restaurante marino que tiene productos por temporada, el restaurante cobra en dolares, soles y euros con un tipo de cambio fijo, ademas el administrador indica que mozo es responsable de cada mesa, cuando se va a usar el sistema tiene que verificar que tipo de usuario es y de acuerdo a ello mostrarle o la pagina administrativa o la pagina de mesero con las mesas asignadas, solamente el administrador puede ingresar productos, editarlos o inhabilitarlos, al finalizar cada orden se tiene que visualizar la nota de pedido en una tabla.
Se requiere crear un tablero Kanban en trello.com y definir las historias de usuario para posteriormente crear las tareas.
